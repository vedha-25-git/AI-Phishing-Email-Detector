# 🔐 AI-Powered Phishing Email Detector

## 📌 Project Overview

The AI-Powered Phishing Email Detector is a cybersecurity web application that identifies phishing and spam emails using Machine Learning and rule-based analysis. The system helps users detect potentially malicious emails by analyzing email content, suspicious keywords, and URLs.

---

## 🚀 Features

* AI-based phishing email detection
* Machine Learning classification using Naive Bayes
* AI Confidence Score
* Risk Score Calculation
* Suspicious Keyword Detection
* URL Detection
* Threat Analysis Report
* "Why Was This Flagged?" Explanation
* User-friendly Cybersecurity Dashboard

## ⚙️ How to Run the Project

1. Clone the repository

   git clone <repository-url>

2. Navigate to the project folder

   cd AI-Phishing-Email-Detector

3. Install dependencies

   pip install flask pandas scikit-learn

4. Run the application

   python app.py

5. Open your browser and visit

   http://127.0.0.1:5000
